# Sistema de Reservas (Barbearia/Clínica...)

Projeto desenvolvido para a disciplina de Arquitetura de Aplicações Web.
Este sistema fornece uma API REST em .NET para agendamentos e um frontend em React.

## Estrutura do Projeto
- `/backend`: API em .NET 10 e integração com MongoDB.
- `/frontend`: Aplicação Web SPA em React + Vite.
